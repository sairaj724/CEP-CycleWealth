
# ♻️ CycleWealth Frontend (Prototype)

## 📌 Project Overview

CycleWealth is a web-based platform designed to connect scrap dealers, artisans, recycling startups, and consumers.
This prototype focuses on the **frontend UI** built using React to demonstrate the core workflow and user interface.

---

## 🚀 Tech Stack

* React (Vite)
* JavaScript (JSX)
* CSS (Custom Styling)
* React Router DOM (for navigation)

---

## 📦 Installation & Setup

### 1️⃣ Create Project (if not created)

```bash
npm create vite@latest cyclewealth-frontend
cd cyclewealth-frontend
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Install Required Packages

```bash
npm install react-router-dom
```

---

## ▶️ Run the Project

```bash
npm run dev
```

👉 Open in browser:

```
http://localhost:5173
```
